# SensorSim

This project simulates environmental sensors for temperature and humidity across multiple data center rooms.

## Features
- Dynamic config loading from JSON
- Multi-sensor, multi-location simulation
- Unit display (°C, %RH)
- Validation and anomaly detection
- Results stored in `sensors.txt` or database

## How to Run
1. Place sensor configs in `ConfigJSON/`
2. Build and run the project
3. View results in console and output file
